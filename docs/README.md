# Screenshots

Drop screenshots here so they render in the main README:

- `trail-map.png` — the main map with a trail + hotspots (desktop)
- `mobile.png` — the mobile view, ideally with the Patterns panel open
- `report.png` — `report.html` rendered

**How to capture clean ones (no personal data):**

1. Temporarily copy `sample-locations.json` over `locations.json`
   (or add a few fake points by hand).
2. Open `tracker.html` / `report.html` in your browser.
3. Screenshot, crop, and save into this folder with the names above.
4. Restore an empty `locations.json` (`[]`) before committing.

⚠️ Never commit screenshots that show a real device's location history.
